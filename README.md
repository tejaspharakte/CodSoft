# C++ Mini Projects

This repository contains beginner-level C++ console projects.

## Projects

1. Guessing Game  
   - A number guessing game using loops and conditions.

2. Simple Calculator  
   - Performs basic arithmetic operations using functions.

3. Tic Tac Toe  
   - Two-player console-based Tic Tac Toe game.

## How to Run

Compile using:

g++ filename.cpp -o output
./output
